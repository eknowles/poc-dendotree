UX Notes

This is a proof of concept that hopes to discover a better way to mass assign items to tree leaf elements, and also create a nice tree

SKUs

A SKU contains its name and group or (decision level), it also has a selected state (true or false).

  • SKU selection can be toggled by clicking on them
  • Multi selection is done by selecting one element and holding SHIFT and selecting another
  • Deselecting and item from multi can be done by holding ALT and click

Once selecting has been made, the user should be able to drag and drop the items into a leaf
